script介绍

介绍

<script> 标签用于在 HTML 文档中嵌入或引用 JavaScript 代码。可以将 JavaScript 代码直接包含在 <script> 标签内部，或通过 src 属性引用外部 JavaScript 文件。

属性

key：src

value：URL

介绍：指定外部 JavaScript 文件的路径。


key：type

value：MIME类型

介绍：指定脚本的 MIME 类型。默认值是 text/javascript。对于模块化脚本，使用 module。


key：async

value：无

介绍：如果存在，表示脚本应该异步加载。仅对外部脚本文件有效。


key：defer

value：无

介绍：如果存在，表示脚本在文档解析完毕后执行。仅对外部脚本文件有效。


key：integrity

value：加密摘要

介绍：允许浏览器验证脚本的完整性，确保文件未被篡改。


key：nomodule

value：无

介绍：如果存在，表示脚本是经典脚本，而不是模块。仅在 <script type="module">时使用 。


key：referrerpolicy

value：no-referrer | no-referrer-when-downgrade | origin | origin-when-cross-origin | same-origin | strict-origin | strict-origin-when-cross-origin | unsafe-url

介绍：设置当请求脚本时，referer 头的策略。


key：crossorigin

value：anonymous | use-credentials

介绍：指定如何处理跨域请求。anonymous 表示不发送用户凭据，use-credentials 表示发送用户凭据。
