Button 专用属性展示

所有类型通用属性

type

类型：可选 button | reset | submit

描述：按钮的类型

写法：<button type="submit">Submit Button</button>

输出：


autofocus

类型：autofocus

描述：页面加载时自动获得焦点

写法：<button autofocus>Autofocus Button</button>

输出：


disabled

类型：disabled

描述：禁用按钮

写法：<button disabled>Disabled Button</button>

输出：


form

类型：form_id

描述：指定按钮所属的表单的ID

写法：<button form="form1">Submit Button</button>

输出：


仅用于type="sumbit"

name

类型：name

描述：按钮的名字(仅用于 type="submit")

写法：<button name="myButton">Named Button</button>

输出：


value

类型：text

描述：按钮的值(仅用于 type="submit")

写法：<button value="ButtonValue">Button with Value</button>

输出：


formaction

类型：URL

描述：指定提交表单的URL (仅用于 type="submit")

写法：<button type="submit" formaction="http://example.com">Submit to Example</button>

输出：


formenctype

类型：可选

描述：指定提交表单的编码类型 (仅用于 type="submit")

写法：<button type="submit" formenctype="multipart/form-data">Submit with Enctype</button>

输出：


formmethod

类型：可选

描述：指定提交表单的方法 (仅用于 type="submit")

写法：<button type="submit" formmethod="post">Submit with POST</button>

输出：


formnovalidate

类型：formnovalidate

描述：提交表单时不进行验证 (仅用于 type="submit")

写法：<button type="submit" formnovalidate>Submit without Validation</button>

输出：


formtarget

类型：可选

描述：指定提交表单后返回结果的打开方式 (仅用于 type="submit")

写法：<button type="submit" formtarget="_blank">Submit to New Tab</button>

输出：


弹出弹窗相关属性

popovertarget

类型：element_id

描述：指定要调用的弹出元素，通过指定元素的 id 来触发该弹出元素

写法：<button popovertarget="popover1">Show Popover</button>

输出：

这是一个弹出元素

popovertargetaction

类型：可选

描述：指定按钮被点击时对弹出元素进行的操作

写法：<button popovertarget="popover1" popovertargetaction="toggle">Toggle Popover</button>

输出：

这是一个弹出元素
