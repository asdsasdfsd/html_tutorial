Input类型展示


type="image"

写法：<input type="image" src="submit.png" alt="Submit">

输出：

介绍：类似于提交按钮，但使用图像作为按钮的外观。


type="submit"

写法：<input type="submit" value="提交">

输出：

介绍：用于提交表单。


type="reset"

写法：<input type="reset" value="重置">

输出：

介绍：重置表单中所有字段为初始值。


type="text"

写法：<input type="text" id="username" name="username">

输出：

介绍：用于接收单行文本输入，例如用户名或简短文本。


type="button"

写法：<input type="button" id="button" name="button" value="Click me">

输出：

介绍：一个普通的按钮，点击时执行自定义的 JavaScript 功能，不会提交表单。


type="checkbox"

写法：

<input type="checkbox" id="color-red" name="color" value="red">

<label for="color-red">红色</label>

<input type="checkbox" id="color-blue" name="color" value="blue">

<label for="color-blue">蓝色</label>

<input type="checkbox" id="color-green" name="color" value="green">

<label for="color-green">绿色</label>

输出：




介绍：允许用户选择多个选项，通常在复选框组中使用。


type="radio"

写法：

男性：<input type="radio" id="male" name="gender" value="male">

女性：<input type="radio" id="female" name="gender" value="female">

输出：
男性：
女性：

介绍：允许用户从多个选项中选择一个，通常在单选框组中使用。


type="number"

写法：<input type="number" id="quantity" name="quantity" min="1" max="10">

输出：

介绍：接收数值输入，通常用于数量或评分等场景。


type="password"

写法：<input type="password" id="password" name="password">

输出：

介绍：用于接收密码输入，输入内容会被隐藏。


type="range"

写法：<input type="range" id="volume" name="volume" min="0" max="100">

输出：

介绍：用于通过滑块选择一个范围内的数值。


type="date"

写法：<input type="date" id="birthday" name="birthday">

输出：

介绍：接收日期输入，用户可以选择一个特定的日期。


type="datetime-local"

写法：<input type="datetime-local" id="meeting-time" name="meeting-time">

输出：

介绍：接收日期和时间输入，用户可以选择特定的日期和时间。


type="month"

写法：<input type="month" id="month" name="month">

输出：

介绍：接收年份和月份输入，用户可以选择一个特定的月份。


type="week"

写法：<input type="week" id="week" name="week">

输出：

介绍：接收年份和周数输入，用户可以选择一个特定的周。


type="time"

写法：<input type="time" id="time" name="time">

输出：

介绍：接收时间输入，用户可以选择一个特定的时间。


type="hidden"

写法：<input type="hidden" id="secret" name="secret" value="hidden-value">

输出：(无显示)

介绍：隐藏字段，用户看不见但会随表单提交到服务器。


type="color"

写法：<input type="color" id="favcolor" name="favcolor">

输出：

介绍：允许用户选择颜色。


type="file"

写法：<input type="file" id="myfile" name="myfile">

输出：

介绍：用于上传文件，用户可以选择本地文件。


type="search"

写法：<input type="search" id="search" name="search" placeholder="Search..."></input>

输出：

介绍：用于搜索输入，回车或者点击表单提交后会提交给服务器。


type="email"

写法：<input type="email" id="email" name="email" placeholder="example@example.com">

输出：

介绍：接收电子邮件地址输入，可以确保用户输入的格式符合电子邮件地址的规范。


type="tel"

写法：<input type="tel" id="phone" name="phone" placeholder="123-456-7890">

输出：

介绍：接收电话号码输入，适合用于输入格式固定的电话号码。


type="url"

写法：<input type="url" id="website" name="website" placeholder="https://www.example.com">

输出：

介绍：接收网址输入，可以确保用户输入的格式符合网址的规范。
