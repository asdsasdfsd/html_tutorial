Input专有属性

必要属性

Key：type

Value：submit | image | reset | text | button | checkbox | radio | number | password | range | date | datetime-local | month | week | time | hidden | color | file | search | email | tel | url

介绍：定义输入字段的类型。类型展示


Key：name

Value：username

介绍：指定输入字段在表单中的名称，用于标识输入数据。


可选属性

Key：form

Value：form_id

介绍：指定 `<input>` 元素所属的表单，使用表单的 id 属性值。


Key：autocomplete

Value：on | off

介绍：允许浏览器自动填充输入字段的值。


Key：autofocus

Value：无

介绍：页面加载时，自动将焦点设置在输入字段上。


Key：disabled

Value：无

介绍：禁用输入字段，使其不可编辑或提交。


Key：formnovalidate

Value：无

介绍：在提交表单时忽略输入字段的验证。


Key：dirname

Value：inputname.dir

写法：<input type="text" id="username" name="username" dirname="username.dir" dir="rtl">

输出：

介绍：在提交表单时，提交1. username: "输入" 2.username.dir: "ltr" 或 "rtl"


验证相关

Key：size

Value：number

介绍：指定输入字段的可见字符数。


Key：maxlength

Value：number

介绍：指定输入字段允许的最大字符数。


Key：minlength

Value：number

介绍：指定输入字段允许的最小字符数。


Key：multiple

Value：无

介绍：允许多个值，适用于 `<input type="file">`。


Key：pattern

Value：[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}

介绍：指定输入字段的正则表达式模式，用于验证输入


Key：placeholder

Value：占位的字符

介绍：在输入前显示的文本提示。


Key：readonly

Value：无

介绍：指定输入字段为只读，用户无法编辑但可提交。


Key：required

Value：无

介绍：指定输入字段为必填项。


限制于特定 type 的属性

type="file"

Key：accept

Value：file_extension | audio/* | video/* | image/* | media_type

介绍：指定允许上传的文件类型。


type="number" 或日期类

Key：step

Value：number | any

介绍：指定数字或日期的递增或递减步长。


Key：min

Value：number

介绍：指定输入字段允许的最小值。


Key：max

Value：number

介绍：指定输入字段允许的最大值。


type="checkbox" 或 `radio`

Key：checked

Value：无

介绍：指定输入字段是否初始选中。


type="submit" 或 `image`

Key：formaction

Value：submit.php

介绍：指定提交表单时的 URL。


Key：formenctype

Value：application/x-www-form-urlencoded | multipart/form-data | text/plain

介绍：指定在提交表单数据时如何编码。


Key：formmethod

Value：post | get

介绍：指定用于提交表单数据的 HTTP 方法。


Key：formtarget

Value：_blank | _self | _parent | _top | framename

介绍：指定在哪个窗口或框架中显示提交的响应。


type="image"

Key：alt

Value：Submit

介绍：指定图像输入字段的替代文本。


Key：src

Value：submit.png

介绍：指定图像的 URL。


Key：height

Value：40

介绍：指定图像的高度。


Key：width

Value：120

介绍：指定图像的宽度。
