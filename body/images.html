图片

<img>

用于在网页中嵌入图像。

  • alt: 为图像指定替代文本，图像无法加载时显示。
  • src: 指定图像的路径。
  • width 和 height: 指定图像的宽度和高度。
  • loading: 可选，指定图像的加载方式。
  • srcset: 可选，指定在不同设备像素密度和视口尺寸下使用的图像文件列表。

<map>

用于定义图像映射，使图像的不同区域可点击。

  • name: 定义映射的名称，与 <img> 的 usemap 属性关联。

<area>

定义图像映射中的可点击区域。

  • alt: 指定区域的替代文本。
  • coords: 指定区域的坐标。
  • href: 指定区域的超链接目标。
  • 其他属性如 download, target, media等。

<canvas>

用于在网页上绘制图形、动画等。

  • width 和 height: 指定画布的宽度和高度。

<figure>和 <figcaption>

<figure> 用于指定独立的内容，如图表、插图等。

<figcaption> 为 <figure> 定义标题。

<picture>

用于定义包含多个图片的容器，结合 srcset 和 sizes属性，实现响应式图像。

<svg>

用于定义可缩放的矢量图形。

  • SVG 使用数学描述定义图形，可以无损缩放。
  • 支持丰富的图形元素和属性定义。
  • 可以与 JavaScript 结合实现交互效果和动画。
  • 支持使用 CSS 进行样式化。
