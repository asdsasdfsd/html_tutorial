窗口事件

  • onafterprint: 指定在文档打印完成后执行的脚本。
  • onbeforeprint: 指定在文档开始打印之前执行的脚本。
  • onbeforeunload: 指定在文档即将被卸载（例如关闭页面）时执行的脚本。
  • onerror: 指定在文档或资源加载过程中发生错误时执行的脚本。
  • onhashchange: 指定在 URL 的锚部分（hash）发生变化时执行的脚本。
  • onload: 指定在页面加载完成后执行的脚本。
  • onmessage: 指定在消息触发时执行的脚本（通常用于跨文档消息）。
  • onoffline: 指定在浏览器开始工作离线时执行的脚本。
  • ononline: 指定在浏览器开始工作在线时执行的脚本。
  • onpagehide: 指定在用户离开页面时执行的脚本。
  • onpageshow: 指定在用户导航到页面时执行的脚本。
  • onpopstate: 指定在窗口历史记录发生变化时执行的脚本（例如通过浏览器前进/后退按钮）。
  • onresize: 指定在浏览器窗口被调整大小时执行的脚本。
  • onstorage: 指定在 Web 存储区域更新时执行的脚本（如 localStorage 或 sessionStorage）。
  • onunload: 指定在页面被卸载时（或浏览器窗口关闭时）执行的脚本。
